# 🌐 Portify – Your Internet, Perfected

Portify is a modern, minimal, and powerful web dashboard that organizes your favorite websites into one clean interface.

🔗 Live: https://eisatopon.github.io

---

## 🚀 Features

- ⚡ Fast and lightweight
- 🎯 One-click access to your favorite sites
- 🌙 Modern dark UI
- ⭐ Save and manage favorites
- 🔍 Smart search
- 📱 Fully responsive
- 🧠 Clean UX – no clutter

---

## 🧩 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage

---

## 💡 Vision

"Your internet in one place."

Portify aims to replace chaotic browsing with a structured, beautiful experience.
